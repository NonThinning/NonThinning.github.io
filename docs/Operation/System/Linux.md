## Command

- `ls` 返回当前目录

```bash
(base) qiao@DESKTOP-9NGQK8T:~$ ls
Miniforge3-Linux-x86_64.sh  miniforge3
```

- `tree` 返回目录树

```bash
sudo apt install tree
```

- `pwd` 查看路径

```bash
(base) qiao@DESKTOP-9NGQK8T:~$ pwd
/home/qiao
```

- `mc` 文件管理

```bash
sudo apt install mc
```
